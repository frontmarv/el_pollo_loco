# El Pollo Loco

_El Pollo Loco_ is a JavaScript side-scrolling Jump 'n' Run game built primarily with HTML, CSS, and JavaScript. Guide your character through exciting levels, avoid obstacles, and take on the wild Pollo!

## Features

- Classic platformer mechanics: running, jumping, and dodging enemies.
- Engaging enemy AI and challenging obstacles.
- Crisp graphics with responsive controls.
- Built using only web technologies—no external frameworks required!

## How to Play

- Use your keyboard's arrow keys or on-screen controls to move your character.
- Jump over obstacles and enemies.
- Reach the end of each level to progress.

## Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/frontmarv/el_pollo_loco.git
   ```

2. **Open the game:**

   - Simply open `index.html` in your web browser.
   - No installation or build steps required.
