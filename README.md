# El Pollo Loco

_El Pollo Loco_ is a JavaScript side-scrolling Jump 'n' Run game built with HTML, CSS, and JavaScript. Play your character "Pepe" and take on the wild "pollos"!

## Table of Contents

- [Features](#features)
- [How to Play](#how-to-play)
- [Getting Started](#getting-started)
- [License](#license)

---

## Features

- Classic platformer mechanics: running, jumping, and throwing objects at enemies.
- Nice animations with responsive controls.
- Built using only web technologies—no external frameworks required!


## How to Play

- Use your keyboard or on-screen controls to move your character.
- Jump fight all enemies to win!

## Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/frontmarv/el_pollo_loco.git
   ```

2. **Open the game:**

   - Simply open `index.html` in your web browser.
   - No installation or build steps required.
